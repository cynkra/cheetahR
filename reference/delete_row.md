# Delete a row from a cheetah grid

Deletes a row from a cheetah grid widget without redrawing the entire
widget.

## Usage

``` r
delete_row(proxy, row_index)
```

## Arguments

- proxy:

  A proxy object created by [`cheetah_proxy()`](cheetah_proxy.md).

- row_index:

  A numeric value representing the index of the row to delete.
