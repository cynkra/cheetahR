# Add a row to a cheetah table

Adds a new row to a cheetah grid widget without redrawing the entire
widget.

## Usage

``` r
add_row(proxy, data)
```

## Arguments

- proxy:

  A proxy object created by [`cheetah_proxy()`](cheetah_proxy.md).

- data:

  A single row dataframe for the new row in the table
